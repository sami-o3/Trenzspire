# Trenzspire

**Connecting Local Brands with Influencers Who Drive Real Impact.**

Trenzspire is a two-sided marketplace platform that connects **social media influencers** with **local brands and businesses** for marketing collaborations.

The platform enables brands to discover influencers based on **followers, engagement rate, niche, and location**, while influencers can explore **campaign opportunities from local businesses** such as restaurants, gyms, cafes, retail stores, and other service providers.

Trenzspire aims to simplify influencer marketing for **local businesses**, making it easier to run targeted campaigns and collaborate with creators who reach the right audience.

---

# 🚀 Features

### Influencer Features

* Create influencer profiles
* Showcase follower count and engagement rate
* Connect social platforms (Instagram, YouTube)
* Discover brands and campaign opportunities
* Apply to campaigns
* Messaging with brands

### Brand Features

* Create brand/business profiles
* Post marketing campaigns
* Discover influencers based on audience metrics
* Filter creators by niche, location, and engagement
* Collaborate with influencers

### Marketplace Features

* Campaign marketplace
* Influencer discovery and filtering
* Brand discovery
* Collaboration messaging system
* Campaign applications and deal management

# 🛠️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/trenzspire.git
```

Navigate to the project directory:

```
cd trenzspire
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
